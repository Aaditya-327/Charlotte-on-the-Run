## COTC enrichment

Charlotte on the Cheap RSS is parsed in two passes:

```mermaid
flowchart LR
    RSS["RSS\n~100 items"] --> DELTA["delta filter\nnewer than last_run"]
    DELTA --> JUNK["junk regex\nVPN ¬Ј deals ¬Ј coupons"]
    JUNK --> GEM["Gemini 2.5 Flash\nbatch 30 items/call\nno search grounding"]
    GEM --> PARSE["parse + validate\nJSON array вАФ null = non-event"]
    PARSE --> DB[("events.db\nupsert by sig")]
```

Extracted per event: `title` ¬Ј `event_date` ¬Ј `event_time` ¬Ј `venue` ¬Ј `neighborhood` ¬Ј `cost` ¬Ј `description` ¬Ј `category[]` ¬Ј `is_recurring`
